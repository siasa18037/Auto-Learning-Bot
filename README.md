# Auto-Learning-Bot

Developed a Python automation bot using Selenium WebDriver to interact with the Voxy learning platform.

Automated login, lesson selection, and exercise completion workflows with intelligent decision logic.

Implemented handling for multiple activity types (Bubble Game, Grammar Swipe, Spelling, Pronunciation).

Enhanced system stability with error recovery, wait conditions, and randomized actions to simulate human behavior.

Applied threading, randomization, and DOM element detection for efficient autonomous operation.
